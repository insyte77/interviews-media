# Интервью и СМИ

Рабочее пространство для взаимодействия со СМИ: ответы на журналистские запросы, экспертные комментарии, интервью.

## Разделы

- **Запросы СМИ** — входящие вопросы от журналистов и редакций
- **Ответы** — подготовленные экспертные комментарии и интервью
- **Темы** — ключевые темы и нарративы для работы со СМИ

## Структура папок

```
/запросы       — входящие запросы от СМИ
/ответы        — готовые ответы и комментарии
/интервью      — полные интервью
/шаблоны       — шаблоны ответов по темам
```
